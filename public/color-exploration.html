从图片取色

?

Image Color Extraction

上传一张图片（点击虚线框或直接拖拽进来）→ 也可以点下方的敦煌壁画样例快速体验 → 自动分析图片里的主要颜色，告诉你每种颜色对应哪个敦煌色 → 点击色块复制色值，点「+」加入收集篮

看到喜欢的海报、照片、插画，想知道里面用了哪些颜色？上传图片（点击虚线框或拖拽进来），或者直接点击下方的敦煌壁画样例试试。系统会自动识别图片中最主要的 8 种颜色，并匹配到最接近的敦煌六主色（窟白、煌金、铅丹、墨玄、石绿、石青）。点击色块就能复制色值，点「+」可以加入收集篮，之后在页面底部的色票导出区统一管理。

查找敦煌色

?

Color Match

输入一个色值（或者点左边的取色器选颜色）→ 点「查找敦煌色」→ 告诉你这个颜色最像六种敦煌色里的哪一种 → 展示敦煌壁画里这个敦煌色通常和哪些颜色在一起 → 下方色块可以快速试试效果

有一个喜欢的颜色，想知道它在敦煌色里最像哪一个？输入任意色值或使用取色器选色，点击「查找敦煌色」。系统会在六种敦煌主色（窟白、煌金、铅丹、墨玄、石绿、石青）中找到最接近的那个，分析它属于哪种色调（淡色、明亮、灰雅、灰调、浅色），并展示敦煌壁画中这个颜色通常和哪些颜色搭配在一起。你可以点开六色排序看看每个敦煌色离你有多远，也可以把配色方案里的颜色加入收集篮。

色票导出

?

Color Export

在上面四个工具里看到喜欢的颜色就点「+」收进来 → 回到这里统一查看和管理 → 点「加载六主色」可以一键填满敦煌六个基础色 → 导出成 CSS / JSON / PNG 格式，用到你的 PPT、海报、网页里

这是一个跨工具的「颜色收集篮」。在上面任何一个工具（主色搭配、文字可读性、从图片取色、查找敦煌色）中，看到喜欢的颜色点击「+ 加入收集篮」按钮，所有颜色就会汇集到这里。你可以一键加载敦煌六主色作为起点，也可以清空重新收集。收集好后，导出为 CSS 变量（直接粘贴到网页样式里）、JSON 格式（保存和分享）、或者下载为 PNG 色卡图片（插入 PPT 或海报软件）。点击色票右侧的 ✕ 可以移除不想要的颜色。
