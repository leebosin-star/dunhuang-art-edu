# UI/UX 设计规范

## 设计理念

- **文化根基**：源自敦煌壁画，非简单堆砌元素，取其神韵
- **简约高级**：大留白、克制用色、精细排版
- **直观易用**：降低学习成本，让美育融入交互

---

## 敦煌色彩体系

参考敦煌壁画矿物颜料，选取淡雅柔和版本：

| 色名 | 色值 | 用途 |
|------|------|------|
| 月白 | `#F7F4EF` | 页面背景 |
| 藕荷 | `#E8D5D0` | 卡片/区块背景 |
| 浅赭 | `#C9A88D` | 装饰线条、边框 |
| 青绿 | `#8BA4A0` | 主交互色、按钮、链接 |
| 石青 | `#5B7B9A` | 标题、强调文本 |
| 浅金 | `#D4B87A` | 点缀、高亮、图标 |
| 朱砂（淡）| `#C47A6D` | 警示、重点强调 |

> 原则：避免使用生硬的土灰色（如 `#999`、`#666`），所有中性色均带敦煌色彩倾向。

---

## 字体规范

| 用途 | 字体 | 备选 |
|------|------|------|
| 标题 | 思源宋体 / Noto Serif SC | Georgia |
| 正文 | 思源黑体 / Noto Sans SC | system-ui |
| 数字/英文 | Inter | system-ui |

字号比例：基于 16px 基准，使用 Tailwind 默认比例尺。

---

## 组件设计原则

1. **圆角**：使用 `rounded-xl` 和 `rounded-2xl`（呼应藻井圆形）
2. **阴影**：柔和大阴影，避免硬阴影（`shadow-lg` 起，配合低透明度）
3. **间距**：宽松留白，最小间距 16px（`p-4`）
4. **动效**：`transition-all duration-300 ease-out`，柔缓过渡
5. **装饰**：可选使用敦煌纹样（忍冬纹、卷草纹）作为背景暗纹

## 页面结构

```
┌──────────────────────────────────┐
│         导航栏（简洁）            │
├──────────────────────────────────┤
│                                  │
│         主内容区                  │
│    （首页为模块入口卡片）         │
│                                  │
├──────────────────────────────────┤
│         页脚（可选）              │
└──────────────────────────────────┘
```

- 导航：左侧 Logo/项目名，右侧模块入口链接
- 首页：两个大卡片分别通往「个性创作」和「实时互动」
- 内容页：全屏沉浸式体验
