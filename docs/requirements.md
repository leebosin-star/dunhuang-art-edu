# 产品需求文档 (PRD)

## 项目概述

**项目名称**：敦煌色彩美育网站
**项目目标**：通过数字化手段传播敦煌色彩美学，提供个性创作与实时互动两大核心体验。
**交付形式**：独立网页应用
**协作模块**：首页、美育专区、色彩演变（其他同学负责）

---

## 模块一：个性创作 — 参数化算法生成

### 核心理念
利用算法帮助没有设计基础的大众生成高级、不落俗套的敦煌风视觉符号。

### 交互场景：多维参数调节

界面提供简约滑块/控制器，实时驱动算法生成画面：

| 参数 | 对应视觉 | 说明 |
|------|---------|------|
| 风的方向 | 线条曲率 | 控制线条弯曲程度与走向 |
| 时间的痕迹 | 斑驳/颗粒度 | 模拟壁画千年风化的肌理感 |
| 乐律 | 元素复杂密度 | 控制纹样密度与层次 |
| 矿物主色 | 主色调 | 淡石绿、砗磲白等敦煌矿物色 |

### 算法生成内容
- 敦煌几何纹样（对称圆形/藻井结构）
- 莲花变体（参数化花瓣形态）
- 波浪飞天线条（流场驱动的飘逸线条）

### 降级备用方案：组件拼贴
- 保留 Canvas 拖拽图层，架构上解耦
- 若参数化效果过于抽象，可切换为拼贴模式
- 提供预制 SVG 素材：九色鹿、祥云、飘带等敦煌元素
- 支持拖拽、缩放、旋转、图层排序

## 模块二：实时互动 — 个人手势交互

### 核心理念
"人在画中游"——用身体操控敦煌意象，创造沉浸式体验。

### 交互场景 A：飞天舞袖

- 用户面对摄像头挥动手臂
- 屏幕实时渲染丝绸质感的「虚拟飘带」
- 飘带跟随手势轨迹，使用敦煌淡矿物色
- 伴随轻微粒子散落效果（模拟壁画颜料飘散）
- 视觉效果：如丝绸般丝滑，非机械跟随

### 交互场景 B：色彩唤醒

- 画面初始：极简、半透明的浅色壁画轮廓
- 用户用手势「抚摸」或指向某个区域
- 该区域如注入矿物颜料般晕染开色彩
- 色彩方向：浅青绿、淡肉粉等敦煌矿物色
- 效果：渐进式晕染，非瞬间填充

## 非功能性需求

| 类别 | 要求 |
|------|------|
| 性能 | 首屏加载 < 3s，手势识别延迟 < 100ms，Canvas 渲染 ≥ 30fps |
| 浏览器兼容 | Chrome 90+, Edge 90+, Safari 15+ |
| 摄像头 | 需用户授权，提供明确的开/关控制 |
| 响应式 | 桌面端为主，手势模块要求 1280x720+ 视口 |
| 安全 | HTTPS 强制，摄像头权限用户可控 |
